## 🎯 7. 사용자 여정별 페이지 흐름

### 여정 1: 솔루션 탐색형
```
메인 페이지
   ↓ (히어로 "솔루션 보기" 클릭)
원스탑 솔루션 섹션 (스크롤)
   ↓ ("전체 솔루션 보기" 클릭)
원스탑 솔루션 페이지
   ↓ (관심 솔루션 클릭)
개별 솔루션 페이지 (예: AI 영상 관제)
   ↓ (하단 CTA 클릭)
문의하기 페이지
```

### 여정 2: 신뢰 검증형
```
메인 페이지
   ↓ (GNB "회사소개" 클릭)
회사소개 페이지
   ↓ (서브메뉴 탐색)
회사연혁 / 인증 및 특허 / 안전보건 경영방침
   ↓ (메인으로 복귀)
시공사례 섹션 확인
   ↓ ("더보기" 클릭)
시공사례 페이지
   ↓ (프로젝트 카드 클릭)
관련 솔루션 페이지
   ↓ (하단 CTA 클릭)
문의하기 페이지
```

### 여정 3: 실적 중심형
```
메인 페이지
   ↓ (시공사례 섹션 확인)
시공사례 섹션
   ↓ ("더보기" 클릭)
시공사례 페이지
   ↓ (필터링 및 탐색)
관심 프로젝트 확인
   ↓ (솔루션 태그 클릭)
해당 솔루션 페이지
   ↓ (하단 CTA 클릭)
문의하기 페이지
```

### 여정 4: 긴급 지원형 (기존 고객)
```
메인 페이지 또는 직접 URL
   ↓ (GNB "고객지원" > "원격지원" 클릭)
원격지원 페이지
   ↓ (원격지원 주소 클릭 또는 전화)
원격 연결 또는 전화 상담
```
