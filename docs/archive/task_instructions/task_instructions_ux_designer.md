# UX Designer 작업 지시사항

## 작업 개요
renewal_content.md의 업데이트된 콘텐츠를 기반으로 template3의 UX 개선 사항을 검토하고 제안해주세요.

## 주요 검토 사항

### 1. 히어로 섹션 UX 개선
- 4개의 슬라이드 콘텐츠가 명확하게 전달되는지 검토
- 각 슬라이드의 CTA 버튼 배치 및 우선순위 검토
- 슬라이드 전환 애니메이션 효과 제안 (텍스트 슬라이드 업 + 페이드 인, 배경 이미지 줌아웃)

### 2. 메인 페이지 섹션 구성
- 원스탑 산업관리 스마트 솔루션 섹션 추가 필요
- 도입효과 섹션 (본사 통합 시스템 설명)
- 숫자로 보는 신뢰 섹션 (설립연도 2009년, 설치 현장 1,500+, 연구 기간 16년+, 고객사 62+)
- 주요 시공사례 섹션 (가로 3개 반응형 그리드)
- 고객지원 섹션

### 3. 사용자 플로우 개선
- 메인 페이지에서 각 솔루션 페이지로의 자연스러운 이동 경로
- 문의하기/원격지원으로의 전환 유도 최적화
- 시공사례 섹션의 필터링 및 탐색 UX

### 4. 다크 테마 최적화
- 현재 template3의 다크 테마 유지
- 가독성과 시각적 계층 구조 검토
- 포인트 컬러 활용 제안

## 참고 문서
- docs/renewal_content.md (업데이트된 콘텐츠)
- web/template3/index.html (현재 구조)
- web/template3/css/ (현재 스타일)

## 결과물
- UX 개선 제안서 (간단한 텍스트 형식)
- 주요 섹션별 레이아웃 제안
- 사용자 인터랙션 개선 사항
