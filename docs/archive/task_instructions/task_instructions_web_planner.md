# Web Planner 작업 지시사항

## 작업 개요
renewal_content.md를 기반으로 template3의 정보 구조(IA)와 사용자 플로우를 재설계해주세요.

## 주요 작업 사항

### 1. 사이트맵 검증
현재 renewal_content.md의 사이트맵과 template3의 실제 구조를 비교하고 누락된 페이지 확인:
- 메인
- 회사소개 (회사소개, 인사말, 회사연혁, 안전보건 경영방침과 목표, 오시는 길)
- 제품 솔루션 (원스탑 산업관리 스마트 솔루션, AI 영상 방송 관제 시스템, 스마트 건설안전 플랫폼, 타워크레인 통합 안전 시스템, 스마트 출입통제 시스템, 스마트 환경센서 시스템)
- 시공사례
- 고객지원 (원격지원, 문의하기)

### 2. 정보 구조 최적화
- 각 페이지의 콘텐츠 계층 구조 정리
- 네비게이션 메뉴 구조 검토 (현재 GNB와 renewal_content.md 비교)
- 서브페이지 공통 요소 정의 (문의 유도 CTA, Top 버튼)

### 3. 사용자 플로우 설계
- 주요 사용자 시나리오별 페이지 이동 경로
  - 솔루션 탐색 → 상세 정보 → 문의
  - 회사 정보 확인 → 신뢰도 검증 → 문의
  - 시공사례 확인 → 관련 솔루션 → 문의

### 4. 콘텐츠 우선순위
- 메인 페이지 섹션 순서 최적화
- 각 솔루션 페이지의 정보 제공 순서
- CTA 배치 전략

### 5. 누락된 페이지 목록
다음 페이지들이 필요하지만 현재 없는 것으로 보입니다:
- 원스탑 산업관리 스마트 솔루션 페이지
- 스마트 건설안전 플랫폼 페이지
- 안전보건 경영방침과 목표 페이지
- 시공사례 페이지

## 참고 문서
- docs/renewal_content.md (전체 콘텐츠 구조)
- web/template3/index.html (현재 메인 페이지)
- web/template3/ 하위 페이지들

- 정보 구조도 (IA)
- 사용자 플로우 다이어그램 (텍스트 형식)
- 누락된 페이지 목록 및 우선순위
- 네비게이션 구조 개선 제안
