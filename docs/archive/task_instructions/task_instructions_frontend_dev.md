## 기술 요구사항

### 애니메이션 효과
- 히어로 섹션: 텍스트 슬라이드 업 + 페이드 인 (0.2초씩 딜레이)
- 히어로 배경: 1.1배에서 1.0배로 줌아웃 효과
- 스크롤 애니메이션: GSAP ScrollTrigger 활용

### 반응형 디자인
- 데스크탑: 가로 3-4개 그리드
- 태블릿: 가로 2개 그리드
- 모바일: 가로 1개 그리드

### 다크 테마 유지
- 현재 template3의 다크 테마 컬러 스킴 유지
- 포인트 컬러 활용
