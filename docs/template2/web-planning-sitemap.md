## 1. 사이트맵 개요

### 1.1 구조 원칙
- **단순성**: 최대 2단계 깊이 (1Depth, 2Depth)
- **명확성**: 사용자가 이해하기 쉬운 메뉴명
- **솔루션 중심**: 4대 핵심 솔루션 명확히 구분
- **확장성**: 향후 솔루션 추가 고려

- **1Depth 메뉴**: 5개
- **2Depth 페이지**: 12개
- **총 페이지**: 13개 (메인 포함)

## 5. URL 네이밍 규칙

### 5.1 URL 구조 원칙
- **소문자 사용**: 모든 URL은 소문자
- **하이픈 구분**: 단어 구분은 하이픈(-) 사용
- **의미 있는 이름**: URL만 봐도 페이지 내용 파악 가능
- **짧고 명확**: 불필요한 단어 제거
- **확장자 명시**: .html 확장자 사용 (정적 사이트)

### 5.2 URL 패턴

```
메인: /
회사소개: /about/{page-name}.html
솔루션: /solutions/{solution-name}.html
고객지원: /support/{page-name}.html
```

### 5.3 URL 예시

```
✅ 좋은 예:
/about/company.html
/solutions/smart-safety.html
/support/contact.html

❌ 나쁜 예:
/about/CompanyInfo.html (대문자 사용)
/solutions/smart_safety.html (언더스코어 사용)
/support/contactus.html (단어 구분 없음)
```

## 7. 브레드크럼 (Breadcrumb) 구조

### 7.1 브레드크럼 패턴

```
홈 > 1Depth > 2Depth

예시:
홈 > 회사소개 > 회사연혁
홈 > 현장 안전 플랫폼 > 스마트 현장안전 시스템
홈 > 고객지원 > 문의하기
```

### 7.2 브레드크럼 표시 규칙
- 메인 페이지: 브레드크럼 없음
- 2Depth 페이지: 홈 > 1Depth > 2Depth
- 구분자: > (꺾쇠)
- 현재 페이지: 링크 없음, 굵게 표시

## 9. SEO 최적화 전략

### 9.1 페이지별 타겟 키워드

| 페이지 | 주요 키워드 | 보조 키워드 |
|--------|------------|------------|
| 메인 | 건설 현장 안전, 스마트 현장 관리 | 현장 솔루션, 안전 시스템, 아이티로그 |
| 스마트 현장안전 | 현장 안전 시스템, 안전교육 시스템 | 방문자 관리, 현장 출입, 안전 관리 |
| 타워크레인 | 타워크레인 안전, 크레인 통합안전 | 타워크레인 관제, 건설 크레인 |
| AI 영상 관제 | AI 영상 관제, 지능형 CCTV | 현장 모니터링, 영상 분석, 스마트 관제 |
| 환경센서 | 스마트 환경센서, 미세먼지 측정 | 풍향풍속, 현장 환경, 환경 모니터링 |
| 출입통제 | 안면인식 출입, 차량 관제 | 출입 통제, 현장 보안, 스마트 출입 |

### 9.2 메타 태그 전략
- **Title**: 50-60자 이내, 주요 키워드 포함
- **Description**: 150-160자 이내, 행동 유도 문구 포함
- **Keywords**: 5-7개 핵심 키워드
