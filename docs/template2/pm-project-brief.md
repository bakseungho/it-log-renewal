## 1. 프로젝트 배경

### 1.1 회사 개요
- **회사명**: (주)아이티로그
- **비즈니스**: 건설 현장 제품 솔루션 설치 및 시공
- **주요 고객**: 건설사, 시공사, 공공기관
- **핵심 가치**: 안전, 기술력, 신뢰, 지속가능성

### 1.2 리뉴얼 필요성
- 건설 현장 안전에 대한 규제 강화 및 시장 수요 증가
- 스마트 건설 기술 트렌드에 맞춘 브랜드 이미지 강화 필요
- B2B 고객의 의사결정을 지원하는 명확한 정보 구조 필요
- 모바일 환경에서의 현장 담당자 접근성 개선
- 솔루션별 체계적인 정보 제공으로 영업 효율성 향상

### 1.3 프로젝트 비전
"건설은 안전하게! 안전은 스마트하게!"라는 슬로건 아래, 기술과 사람을 잇는 혁신적인 안전 체계를 제공하는 신뢰할 수 있는 파트너로 자리매김

## 2. 프로젝트 목표

### 2.1 비즈니스 목표
1. **브랜드 포지셔닝 강화**: 스마트 건설 안전 솔루션 선도 기업 이미지 구축
2. **리드 생성 증대**: 문의 전환율 35% 향상
3. **정보 접근성 개선**: 솔루션 정보 탐색 시간 60% 단축
4. **고객 신뢰도 향상**: 실적 및 사례 중심의 신뢰 구축
5. **시장 경쟁력 강화**: 경쟁사 대비 차별화된 온라인 경험 제공

### 2.2 사용자 목표
1. **빠른 솔루션 탐색**: 필요한 솔루션을 3클릭 이내 접근
2. **명확한 정보 제공**: 시스템 구성 및 적용 사례 명확히 이해
3. **신속한 문의**: 간편한 문의 프로세스로 빠른 상담 연결
4. **현장 활용**: 모바일에서 솔루션 정보 즉시 확인
5. **원격지원 접근**: 기존 고객의 편리한 원격지원 서비스 이용

### 2.3 기술 목표
1. **성능 최적화**: 페이지 로딩 속도 2.5초 이내
2. **반응형 구현**: 모든 디바이스에서 최적화된 경험
3. **유지보수성**: 정적 웹사이트 구조로 쉬운 업데이트
4. **접근성 준수**: WCAG 2.1 AA 수준 달성
5. **SEO 최적화**: 주요 키워드 검색 순위 상위 진입

## 4. 타겟 사용자

### 4.1 주요 페르소나

#### 페르소나 1: 구매 의사결정자 (임원/관리자)
- **연령**: 45-60세
- **역할**: 구매 결정권자, 프로젝트 승인자
- **니즈**: 회사 신뢰도, 실적, 안전성 확인
- **행동**: 데스크톱 사용, 상세 정보 검토, 경쟁사 비교
- **목표**: 검증되고 안정적인 솔루션 선정
- **핵심 관심사**: 시공사례, 인증, 고객사

#### 페르소나 2: 현장 담당자 (안전관리자/기술자)
- **연령**: 30-45세
- **역할**: 기술 검토, 현장 적용 가능성 판단
- **니즈**: 시스템 구성도, 기술 사양, 설치 사례
- **행동**: 모바일 사용 빈번, 빠른 정보 확인
- **목표**: 현장에 최적화된 솔루션 찾기
- **핵심 관심사**: 시스템 개요, 구성도, 적용 사례

#### 페르소나 3: 구매/조달 담당자
- **연령**: 28-40세
- **역할**: 견적 요청, 업체 비교, 계약 진행
- **니즈**: 빠른 문의, 명확한 연락처, 회사 정보
- **행동**: 다양한 디바이스 사용, 효율성 중시
- **목표**: 신속한 업무 처리 및 의사결정 지원
- **핵심 관심사**: 문의하기, 연락처, 회사 개요

### 4.2 사용자 여정

#### 인지 단계
- 검색 엔진 유입 (건설 현장 안전, AI 영상 관제 등)
- 추천 또는 소개를 통한 방문
- 전시회/세미나 후 웹사이트 방문

#### 고려 단계
- 메인 페이지에서 주요 솔루션 확인
- 관심 솔루션 상세 페이지 탐색
- 시공사례 및 시스템 구성도 검토
- 회사 정보 및 인증 확인

#### 결정 단계
- 문의 폼 작성 또는 전화 연락
- 원격지원 서비스 확인 (기존 고객)
- 견적 요청 및 상담 진행

## 5. 핵심 성공 요인

### 5.1 차별화 전략

**1. 4단계 히어로 스토리텔링**
- 슬라이드 1: 브랜드 가치 (슬로건)
- 슬라이드 2: 핵심 기술 (AI/통합안전)
- 슬라이드 3: 현장 맞춤 솔루션 (환경센서)
- 슬라이드 4: 신뢰와 유지보수 (CS)

**2. 솔루션 중심 구조**
- 4대 핵심 솔루션 명확히 구분
- 각 솔루션별 체계적인 정보 제공
- 시스템 구성도를 통한 시각적 이해 지원

**3. 신뢰 구축 요소**
- 숫자로 보는 신뢰 섹션 (설치 횟수, 연구 기간, 고객사 수)
- 주요 고객사 CI 로고
- 시공사례 중심의 실적 제시

**4. 고객 지원 강화**
- 원격지원 서비스 강조
- 365일 전문 엔지니어 지원 메시지
- 간편한 문의 프로세스

### 5.2 B2B 특화 전략

**신뢰 구축**
- 인증 및 특허 전면 배치
- 20년 경험 강조
- 구체적인 시공사례 제공

**전문성 강조**
- 시스템 구성도 및 일러스트
- 기술 용어 사용 (이해하기 쉽게 설명)
- 하위 시스템별 상세 정보

**의사결정 지원**
- 명확한 솔루션 분류
- 적용 분야 명시
- 빠른 문의 경로 제공
