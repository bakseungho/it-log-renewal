# 웹사이트 리뉴얼 프로젝트 요약

## 프로젝트 개요
renewal_content.md의 콘텐츠를 라이트 테마(`web/light/`)에 전체적으로 적용하는 리뉴얼 프로젝트

## 테마
- **라이트 모드 단일 운영**
- 밝은 배경 (#ffffff), 어두운 텍스트 (#1a1a1a)
- Primary 컬러: #0066FF (WCAG AA 준수)
- Pretendard 폰트

## 완료된 작업

### 1. renewal_content.md 콘텐츠 보완
- **숫자로 보는 신뢰**: 설치 현장 1,500+, 연구 기간 16년+
- **시공사례 레이아웃**: 반응형 그리드 (데스크탑 3개, 태블릿 2개, 모바일 1개)
- **고객지원 서브 문구**: "전문 엔지니어가 365일 귀사의 현장을 함께 지킵니다"
- **기술력 상세 설명**: Big Data, Deep Learning, Smart Safety, IoT
- **히어로 섹션 애니메이션**: 텍스트 딜레이 0.2초, 배경 줌아웃 효과

### 2. 전체 페이지 구현 완료
- 메인 페이지 (히어로, 원스탑 솔루션, 도입효과, 숫자로 보는 신뢰, 고객사, 시공사례, 고객지원)
- 솔루션 페이지 6개
- 회사소개 페이지 6개
- 시공사례 페이지
- 고객지원 페이지 2개
- 공통 페이지 (이용약관, 개인정보처리방침, 404)

- 설립: 2009년

- 연구 기간: 16년+

### 히어로 섹션 (4개 슬라이드)
1. 브랜드 가치: "건설은 안전하게! 안전은 스마트하게!"

3. 현장 맞춤 솔루션: "쾌적한 환경부터 철저한 출입통제 솔루션까지"
4. 신뢰와 유지보수: "전국 어디서나, 멈춤 없는 원격지원과 사후관리"

### 제품 솔루션 (6개)
1. 원스탑 산업관리 스마트 솔루션

3. 스마트 건설안전 플랫폼
4. 타워크레인 통합 안전 시스템

## 기술 스택
- HTML5, CSS3, JavaScript (ES6+)
- Swiper.js (히어로 슬라이드)
- GSAP + ScrollTrigger (스크롤 애니메이션)
- 반응형 디자인 (모바일 퍼스트)
- CSS Variables 기반 디자인 시스템
