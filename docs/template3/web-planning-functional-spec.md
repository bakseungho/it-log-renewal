## 1. 기능 개요

### 1.1 핵심 기능
1. 다크 모드 네비게이션 시스템
2. 풀스크린 히어로 슬라이더 (4개)

4. 프리미엄 호버 인터랙션
5. 문의 폼 (이메일 전송)

7. 이미지 라이트박스
8. 반응형 디자인

### 1.2 기술 스택
- HTML5 (시맨틱 태그)
- CSS3 (다크 모드, 그라데이션, 블러, 글로우)
- JavaScript (ES6+)
- GSAP 3.x + ScrollTrigger (CDN)
- Swiper.js (슬라이더)
- 카카오맵 API

## 2. 공통 기능

### 2.1 다크 모드 네비게이션

**기능 ID**: F-001  
**우선순위**: 필수  
**설명**: 모든 페이지에서 일관된 다크 테마 네비게이션

**상세 기능**:

#### 데스크톱 네비게이션
- **초기 상태**:
  - 배경: 투명 또는 반투명 (#0a0a0a, opacity: 0.8)
  - 텍스트: 화이트 (#ffffff)
  - 로고: 화이트 버전
  
- **스크롤 상태** (100px 이상):
  - 배경: 불투명 (#0a0a0a, opacity: 1.0)
  - 블러 효과 (backdrop-filter: blur(10px))
  - 하단 테두리: 1px solid rgba(255,255,255,0.1)

- **호버 상태**:
  - 텍스트: 골드 (#d4af37) 또는 시안 (#00d9ff)
  - 언더라인 애니메이션 (왼쪽→오른쪽, 0.3초)

- **드롭다운 메뉴**:
  - 배경: #1a1a1a
  - 페이드인 애니메이션 (0.3초)
  - 2Depth 메뉴 표시

#### 모바일 네비게이션
- **햄버거 아이콘**:
  - 우측 상단 배치
  - 3줄 아이콘 (화이트)
  - 클릭 시 X 아이콘으로 변환

- **전체 화면 메뉴**:
  - 배경: #0a0a0a (opacity: 0.98)
  - 슬라이드 인 애니메이션 (우측→좌측)
  - 1Depth + 2Depth 메뉴 표시
  - 닫기 버튼 (X)

**기술 구현**:
```javascript
// 스크롤 이벤트
window.addEventListener('scroll', () => {
  if (window.scrollY > 100) {
    nav.classList.add('scrolled');
  } else {
    nav.classList.remove('scrolled');
  }
});
```

## 3. 메인 페이지 기능

### 3.1 풀스크린 히어로 슬라이더

**기능 ID**: F-003  
**우선순위**: 필수  
**설명**: 4개 슬라이드로 구성된 프리미엄 히어로 섹션

**슬라이드 구성**:
1. 브랜드 가치 (슬로건)
2. 핵심 기술 (AI/통합안전)
3. 현장 맞춤 솔루션 (환경센서)
4. 신뢰와 유지보수 (CS)

**기능 상세**:

#### 자동 재생
- 간격: 5초
- 전환 효과: 페이드 (1초)
- 무한 반복

#### 수동 제어
- 좌우 화살표 버튼
- 하단 인디케이터 (점)
- 스와이프 제스처 (모바일)

#### 인디케이터
- 위치: 하단 중앙
- 스타일: 원형 점 (골드/시안)
- 현재 슬라이드: 불투명, 크기 증가
- 다른 슬라이드: 반투명, 작은 크기
- 클릭 시: 해당 슬라이드 이동

**기술 구현** (Swiper.js):
```javascript
const heroSwiper = new Swiper('.hero-swiper', {
  effect: 'fade',
  autoplay: {
    delay: 5000,
    disableOnInteraction: false,
  },
  pagination: {
    el: '.swiper-pagination',
    clickable: true,
  },
  navigation: {
    nextEl: '.swiper-button-next',
    prevEl: '.swiper-button-prev',
  },
});
```

```javascript

    start: 'top 80%',

  duration: 0.8,
  stagger: 0.1,
```

### 5.2 문의하기 폼

**기능 ID**: F-013  
**우선순위**: 필수  
**설명**: 프리미엄 문의 폼 (다크 모드)

**폼 필드**:
1. 이름 (필수, text)
2. 회사명 (필수, text)
3. 연락처 (필수, tel)
4. 이메일 (필수, email)
5. 관심 솔루션 (선택, select)
6. 문의 내용 (필수, textarea)
7. 개인정보 처리방침 동의 (필수, checkbox)

**유효성 검사**:
- 필수 필드 확인
- 이메일 형식 검증
- 연락처 형식 검증
- 실시간 에러 메시지 표시

**제출 기능**:
- 버튼: "상담 신청" (골드 배경)
- 로딩 상태 표시
- 성공 메시지: "문의가 접수되었습니다"
- 실패 메시지: "오류가 발생했습니다"

**기술 구현**:
```javascript
form.addEventListener('submit', async (e) => {
  e.preventDefault();
  
  // 유효성 검사
  if (!validateForm()) return;
  
  // 폼 데이터 수집
  const formData = new FormData(form);
  
  // 이메일 전송 (예시)
  try {
    const response = await fetch('/api/contact', {
      method: 'POST',
      body: formData,
    });
    
    if (response.ok) {
      showSuccessMessage();
      form.reset();
    } else {
      showErrorMessage();
    }
  } catch (error) {
    showErrorMessage();
  }
});
```
