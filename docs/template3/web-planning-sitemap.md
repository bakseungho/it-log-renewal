## 4. 네비게이션 설계

### 4.1 글로벌 네비게이션 (GNB)

**다크 모드 스타일**:
- 배경: #0a0a0a (반투명 또는 블러)
- 텍스트: #ffffff
- 호버: 골드(#d4af37) 또는 시안(#00d9ff) 언더라인
- 스크롤 시: 배경 불투명도 증가

**구조**:
```
[로고] 회사소개 | 현장 안전 플랫폼 | 스마트 관제·제어 | 환경·출입관리 | 고객지원
```

**드롭다운 메뉴** (데스크톱):
- 호버 시 2Depth 메뉴 표시
- 다크 배경 (#1a1a1a)
- 부드러운 페이드인 애니메이션

**햄버거 메뉴** (모바일):
- 우측 상단 햄버거 아이콘
- 전체 화면 오버레이 메뉴
- 다크 배경 (#0a0a0a)
- 슬라이드 인 애니메이션

### 4.2 푸터 네비게이션

**다크 모드 스타일**:
- 배경: #000000
- 텍스트: #a0a0a0 (보조), #ffffff (강조)
- 링크 호버: 골드 또는 시안

**구조**:
- 회사 기본정보 (CI, 주소, 전화번호, 팩스, 사업자번호)
- 이용약관
- 개인정보처리방침
- 저작권 표기
- 문의하기 버튼

### 4.3 브레드크럼 (Breadcrumb)

**위치**: 서브 페이지 상단  
**스타일**: Home > 1Depth > 2Depth  
**컬러**: 그레이 (#a0a0a0), 현재 페이지 화이트 (#ffffff)

**예시**:
- `Home > 회사소개 > 회사연혁`
- `Home > 현장 안전 플랫폼 > 스마트 현장안전 시스템`

### 6.1 주요 사용자 플로우

#### 플로우 1: 솔루션 탐색
```
메인 페이지 → 주요 솔루션 섹션 → 솔루션 카드 클릭 
→ 솔루션 상세 페이지 → 시스템 구성도 확인
```

```
메인 페이지 → 숫자로 보는 신뢰 섹션 → 고객사 섹션 
→ 회사소개 메뉴 → 인증 및 특허 → 문의하기
```

#### 플로우 3: 직접 문의
```
메인 페이지 → 고객문의 섹션 → 문의하기 버튼
```

```
메인 페이지 → 고객지원 메뉴 → 원격지원 
→ 외부 사이트 이동
```

## 7. SEO 구조

### 7.1 메타 태그 전략

**각 페이지별 필수 메타 태그**:
- `<title>`: 페이지별 고유 제목 (60자 이내)
- `<meta name="description">`: 페이지 설명 (160자 이내)
- `<meta name="keywords">`: 주요 키워드
- Open Graph 태그 (소셜 공유)
- Canonical URL

**예시 (메인 페이지)**:
```html
<title>아이티로그 | 건설 현장 스마트 안전 솔루션</title>
<meta name="description" content="AI 영상 관제, 스마트 환경센서, 출입통제 등 건설 현장 안전을 위한 프리미엄 솔루션">
<meta name="keywords" content="건설 안전, AI 영상 관제, 스마트 센서, 출입통제, 타워크레인">
```

### 7.2 구조화된 데이터 (Schema.org)

**Organization Schema** (메인 페이지):
```json
{
  "@context": "https://schema.org",
  "@type": "Organization",
  "name": "(주)아이티로그",
  "url": "https://www.itlog.co.kr",
  "logo": "https://www.itlog.co.kr/images/logo.png",
  "description": "건설 현장 스마트 안전 솔루션"
}
```

**Product Schema** (솔루션 페이지):
```json
{
```
