# UX 와이어프레임 (Template3)

## 문서 개요

**프로젝트명**: (주)아이티로그 홈페이지 리뉴얼 (Template3)  
**작성일**: 2024  
**작성자**: UX Designer  
**버전**: 1.0  
**디자인 테마**: Dark & Premium

---

## 1. 와이어프레임 개요

### 1.1 목적

Template3의 주요 페이지에 대한 레이아웃 구조와 콘텐츠 배치를 시각적으로 정의합니다. 다크 모드 프리미엄 디자인의 특징인 풀스크린 레이아웃, 대형 타이포그래피, 넉넉한 여백을 반영합니다.

### 1.2 와이어프레임 범위

**주요 페이지** (상세 와이어프레임):
1. 메인 페이지 (index.html)
2. 회사소개 (about/company.html)
3. 솔루션 상세 (solutions/smart-safety.html)
4. 문의하기 (support/contact.html)

**기타 페이지** (간략 구조):
- CEO 인사말, 회사연혁, 인증/특허, 오시는 길
- 기타 솔루션 페이지
- 원격지원

- 높이: 80px

## 8. 인터랙션 플로우

### 8.1 메인 페이지 스크롤 플로우

```
1. 히어로 섹션 (100vh)
   ↓ 스크롤
2. 히어로 슬라이드 자동 전환 (5초)
   ↓ 스크롤
3. 주요 솔루션 섹션 페이드인
   ↓ 스크롤
4. 숫자 카운트업 애니메이션
   ↓ 스크롤
5. 고객사 로고 페이드인
   ↓ 스크롤
6. 시공사례 이미지 갤러리
   ↓ 스크롤
7. 고객문의 섹션
   ↓ 스크롤
8. 푸터
```

### 8.2 솔루션 카드 호버 플로우

```
1. 마우스 오버
   ↓
2. 카드 스케일(1.05) + 골드 글로우
   ↓
3. 아이콘 회전 또는 스케일
   ↓
4. "자세히 →" 텍스트 언더라인
   ↓
5. 클릭 → 솔루션 상세 페이지 이동
```

### 8.3 문의 폼 제출 플로우

```
1. 폼 작성
   ↓
2. 필드 포커스 → 골드 테두리
   ↓
3. 유효성 검사 (실시간)
   ↓
4. 개인정보 동의 체크
   ↓
5. "상담 신청" 버튼 클릭
   ↓
6. 로딩 애니메이션
   ↓
7. 성공 메시지 표시
```
