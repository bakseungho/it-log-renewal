## 1. 기능 요구사항

### 1.1 공통 요구사항

#### FR-001: 다크 모드 네비게이션
- **우선순위**: 필수
- **설명**: 모든 페이지에서 일관된 다크 테마 네비게이션 제공
- **상세 요구사항**:
  - 배경색: #0a0a0a (반투명 또는 블러 효과)
  - 텍스트: #ffffff
  - 호버 시: 골드(#d4af37) 또는 시안(#00d9ff) 언더라인
  - 스크롤 시 배경 불투명도 증가
  - 모바일: 다크 테마 햄버거 메뉴
- **검수 기준**: 모든 디바이스에서 가독성 확보, 스크롤 인터랙션 정상 작동

#### FR-002: 다크 모드 푸터
- **우선순위**: 필수
- **설명**: 프리미엄 느낌의 다크 테마 푸터
- **상세 요구사항**:
  - 배경색: #000000
  - 텍스트: #a0a0a0 (보조), #ffffff (강조)
  - 회사 정보, 이용약관, 개인정보처리방침, 문의하기
  - 골드 또는 시안 포인트 컬러 활용
- **검수 기준**: 가독성 확보, 링크 정상 작동

#### FR-003: 반응형 디자인 (다크 모드)
- **우선순위**: 필수
- **설명**: 모든 디바이스에서 최적화된 다크 모드 경험
- **상세 요구사항**:
  - 모바일: 320px~767px
  - 태블릿: 768px~1023px
  - 데스크톱: 1024px 이상
  - 다크 모드 색상 일관성 유지
- **검수 기준**: 모든 브레이크포인트에서 레이아웃 정상, 가독성 확보

### 1.2 메인 페이지 요구사항

#### FR-004: 풀스크린 히어로 슬라이더 (다크 모드)
- **우선순위**: 필수
- **설명**: 4개 슬라이드로 구성된 프리미엄 히어로 섹션
- **상세 요구사항**:
  - 높이: 100vh (풀스크린)
  - 배경: 다크 그라데이션 또는 고해상도 이미지
  - 텍스트: 화이트(#ffffff), 골드(#d4af37) 강조
  - 타이포그래피: 60px~120px (데스크톱), 32px~48px (모바일)
  - 자동 재생: 5초 간격
  - 전환 효과: 페이드 (1초)
  - 네비게이션: 하단 인디케이터 (골드/시안)
  - 스와이프 제스처 지원
- **슬라이드 내용**:
  1. 브랜드 가치 (슬로건)
  2. 핵심 기술 (AI/통합안전)
  3. 현장 맞춤 솔루션 (환경센서)
  4. 신뢰와 유지보수 (CS)
- **검수 기준**: 부드러운 전환, 가독성 확보, 모바일 정상 작동

#### FR-005: 주요 솔루션 섹션 (다크 모드)
- **우선순위**: 필수
- **설명**: 4개 솔루션 카드 그리드
- **상세 요구사항**:
  - 배경: #1a1a1a (카드)
  - 레이아웃: 2x2 그리드 (데스크톱), 1열 (모바일)
  - 호버 효과: 스케일(1.05) + 글로우 효과 + 골드/시안 테두리
  - 아이콘: 골드 또는 시안
  - 텍스트: 화이트(제목), 그레이(설명)
  - 스크롤 애니메이션: 페이드인 + 슬라이드업
- **솔루션 목록**:
  1. 지능형 영상 관제
  2. 스마트 환경센서
  3. 스마트 출입통제
  4. 스마트 현장안전
- **검수 기준**: 호버 효과 부드러움, 링크 정상 작동

#### FR-006: 숫자로 보는 신뢰 섹션 (다크 모드)
- **우선순위**: 필수
- **설명**: 실적 데이터 시각화
- **상세 요구사항**:
  - 배경: #0a0a0a
  - 숫자: 큰 사이즈(48px~72px), 골드 또는 시안
  - 레이블: 화이트, 16px~18px
  - 카운트업 애니메이션 (스크롤 트리거)
  - 3-4개 지표 (설치 횟수, 연구 기간, 고객사 수 등)
- **검수 기준**: 카운트업 애니메이션 정상, 가독성 확보

#### FR-007: 고객사 섹션 (다크 모드)
- **우선순위**: 필수
- **설명**: 주요 고객사 CI 로고 나열
- **상세 요구사항**:
  - 배경: #1a1a1a
  - 로고: 화이트 또는 그레이스케일 처리
  - 레이아웃: 그리드 또는 슬라이더
  - 호버 효과: 밝기 증가 또는 컬러 복원
- **검수 기준**: 로고 가독성, 정렬 정상

#### FR-008: 주요 시공사례 섹션 (다크 모드)
- **우선순위**: 필수
- **설명**: 대표 프로젝트 실적 리스트
- **상세 요구사항**:
  - 배경: #0a0a0a
  - 카드: #1a1a1a, 이미지 + 제목 + 설명
  - 호버 효과: 이미지 스케일 + 오버레이 페이드인
  - 레이아웃: 3열 그리드 (데스크톱), 1열 (모바일)
  - 클릭 시: 해당 솔루션 페이지 이동
- **검수 기준**: 이미지 로딩 정상, 링크 작동

#### FR-009: 고객문의 섹션 (다크 모드)
- **우선순위**: 필수
- **설명**: CS 정보 및 CTA 버튼
- **상세 요구사항**:
  - 배경: #1a1a1a 또는 그라데이션
  - 텍스트: 화이트
  - CTA 버튼: 골드 또는 시안 배경, 화이트 텍스트
  - 호버 효과: 밝기 증가 + 스케일
  - 버튼: "원격지원", "문의하기"
- **검수 기준**: 버튼 정상 작동, 가독성 확보

### 1.3 서브 페이지 요구사항

  - 타이포그래피: 화이트(제목), 그레이(본문)
  - 스크롤 애니메이션: 페이드인
- **검수 기준**: 가독성, 레이아웃 정상

#### FR-012: 인증 및 특허 갤러리 (다크 모드)

  - 제출 버튼: 골드 또는 시안 배경
- **검수 기준**: 폼 제출 정상, 유효성 검사

#### FR-016: GSAP 스크롤 애니메이션

  - 모바일: 비활성화 (성능)

#### FR-018: 호버 인터랙션 (다크 모드)
- **우선순위**: 필수

  - 버튼: 스케일(1.05) + 밝기 증가
  - 카드: 스케일(1.05) + 글로우 효과 + 골드/시안 테두리

  - 지속 시간: 0.3~0.4초

## 3. 디자인 요구사항

### 3.1 컬러 시스템 (다크 모드)

#### DR-001: 컬러 팔레트
- **우선순위**: 필수
- **Primary Dark**: #0a0a0a (메인 배경)
- **Secondary Dark**: #1a1a1a (카드, 섹션)
- **Accent Gold**: #d4af37 (프리미엄 포인트)
- **Accent Cyan**: #00d9ff (기술력 강조)
- **Text White**: #ffffff (주요 텍스트)
- **Text Gray**: #a0a0a0 (보조 텍스트)
- **검수 기준**: 모든 페이지에서 일관된 컬러 사용

#### DR-002: 색상 대비
- **우선순위**: 필수
- **요구사항**:
  - 텍스트 대비: 4.5:1 이상
  - UI 요소 대비: 3:1 이상
  - 골드/시안 포인트 컬러 가독성 확보
