# Template3 프로젝트 문서

## 개요

**프로젝트명**: (주)아이티로그 홈페이지 리뉴얼 (Template3)  
**디자인 테마**: Dark & Premium (다크 모드 기반 프리미엄)  
**참고 사이트**: https://www.dwce.co.kr/  
**작성일**: 2024

---

## 디자인 컨셉

### 핵심 키워드
- **Dark Mode**: 다크 모드 기반의 세련된 디자인
- **Premium**: 고급스럽고 전문적인 이미지
- **Impact**: 대형 비주얼과 임팩트 있는 타이포그래피
- **Interactive**: 스크롤 기반의 인터랙티브한 경험
- **Minimal**: 미니멀하고 모던한 레이아웃

### Template 비교

| 구분 | Template1 | Template2 | Template3 |
|------|-----------|-----------|-----------|
| **테마** | 전통적 & 안정적 | 트렌디 & 심플 | 다크 & 프리미엄 |
| **컬러** | 따뜻한 톤 | 차가운 블루 | 다크 + 골드/시안 |
| **분위기** | 신뢰감 | 혁신적 | 고급스러움 |
| **타겟** | 보수적 고객 | 젊은 의사결정자 | 프리미엄 고객 |

## 주요 특징

### 디자인 차별화
- **다크 모드 기반**: 검은색 배경에 화이트/골드 텍스트
- **대형 비주얼**: 풀스크린 이미지와 영상
- **임팩트 타이포**: 큰 사이즈의 굵은 폰트
- **스크롤 인터랙션**: 패럴랙스, 페이드, 스케일 효과
- **프리미엄 컬러**: 골드, 시안, 화이트 포인트

### 기술 스택
- HTML5 (시맨틱 태그)
- CSS3 (다크 모드, 그라데이션, 블러)
- JavaScript (ES6+)
- GSAP (고급 애니메이션)
- ScrollTrigger (스크롤 기반 인터랙션)
