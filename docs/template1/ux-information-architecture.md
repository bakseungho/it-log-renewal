# UX 정보 구조 및 페이지 맵

## 1. 사이트 구조 개요

### 1.1 구조 원칙
- **단순성**: 최대 3단계 깊이로 제한
- **명확성**: 직관적인 카테고리 명명
- **확장성**: 향후 콘텐츠 추가 고려
- **사용자 중심**: 사용자의 정보 탐색 경로 최적화

### 1.2 전체 페이지 수
- **1Depth 메뉴**: 5개 (회사소개, 현장 안전 플랫폼, 스마트 관제·제어, 환경·출입관리, 고객지원)
- **2Depth 페이지**: 17개
- **총 페이지**: 23개 (메인 포함)

### 1.3 네비게이션 전략
- **글로벌 네비게이션**: 5개 주요 섹션 (솔루션 중심 구조)
- **로컬 네비게이션**: 섹션 내 하위 페이지
- **유틸리티 네비게이션**: 검색, 문의하기
- **푸터 네비게이션**: 전체 사이트맵, 부가 정보

## 3. 주요 페이지 상세 구조

### 3.1 홈페이지 (Home)

#### 목적
- 회사의 첫인상 형성
- 핵심 가치 제안 전달
- 주요 솔루션으로 빠른 유도

#### 섹션 구조
```
1. Hero Section (슬라이드 4개)
   슬라이드 1: 브랜드 가치
   - 서브타이틀: Sustainable Safety Partner
   - 제목: 건설은 안전하게! 안전은 스마트하게!
   - 서브 문구: 기술과 사람을 잇는 아이티로그만의 혁신적인 안전 체계
   - CTA: [견적 문의하기] [솔루션 둘러보기]
   
   슬라이드 2: 핵심 기술 (AI/통합안전)
   - 서브타이틀: Smart AI Surveillance
   - 제목: 사각지대 없는 지능형 AI 영상 관제 시스템
   - CTA: [자세히 보기]
   
   슬라이드 3: 현장 맞춤 솔루션 (환경센서)
   - 서브타이틀: Specialized Field Solution
   - 제목: 쾌적한 환경부터 철저한 출입통제 솔루션까지
   - CTA: [자세히 보기]
   
   슬라이드 4: 신뢰와 유지보수 (CS)
   - 서브타이틀: Reliable Support & CS
   - 제목: 전국 어디서나, 멈춤 없는 원격지원과 사후관리
   - CTA: [원격지원 바로가기]
   
2. 주요 솔루션 섹션
   - 4개 솔루션 카드 (지능형 영상 관제, 스마트 환경센서, 스마트 출입통제, 스마트 현장안전)
   - 각 카드: 아이콘/이미지 + 제목 + 간단 설명 + [자세히 보기]
   
3. 숫자로 보는 신뢰 섹션
   - 설치 횟수: 1,247+
   - 연구 기간: 20년+
   - 고객사 수: 523+
   - 카운터 애니메이션
   
4. 주요 고객사
   - 로고 그리드
   - 신뢰도 강화
   
5. 주요 시공사례 섹션
   - 3-4개 대표 사례
   - 슬라이더 형태
   
6. 고객문의 섹션
   - CS 정보 (전화번호, 운영시간)
   - [원격지원] 버튼
   - [문의하기] 버튼
```

#### 사용자 플로우
```
방문 → Hero 메시지 확인 → 관심 솔루션 클릭 → 상세 정보 탐색 → 문의/다운로드
```

### 3.2 회사소개 (About)

#### 3.2.1 회사소개
```
- 회사 소개 텍스트
- 기업정보 (소개 문구, 기업 이념, 안전보건 경영방침/목표)
- 사업분야 (주요 비즈니스 영역 요약)
- 주요 고객사 로고
```

#### 3.2.2 CEO 인사말
```
- CEO 사진
- 인사말 전문
- 서명
```

#### 3.2.3 회사연혁
```
- 타임라인 형태 (2010년~2025년)
- 주요 마일스톤
- 연도별 주요 성과
- 시각적 표현 (세로 타임라인)
```

#### 3.2.4 인증 및 특허
```
- 인증서 이미지 그리드
- 특허 목록
- 수상 내역
- 다운로드 가능
```

#### 3.2.5 오시는 길
```
- 지도 (카카오맵 API 임베드)
- 주소 및 연락처
- 대중교통 안내
- 주차 안내
```

### 3.3 솔루션 페이지 (Solutions)

#### 공통 구조 (6개 솔루션 페이지)
```
1. 시스템 개요
   - 솔루션 소개 (2-3 문단)
   - 주요 특징 (3-5개 불릿 포인트)
   - 적용 분야
   
2. 시스템 구성도
   - 구성도 이미지 또는 일러스트
   - 각 구성 요소 설명
   - 기술 사양 (선택)
   
3. 하위 시스템 (해당 시)
   - 탭 또는 아코디언 방식
   - 각 하위 시스템별 개요, 구성도, 특징
   
4. 주요 시공 사례
   - 사례 이미지 3-4개
   - 프로젝트명, 발주처, 기간
   - 주요 성과
   - 라이트박스 확대 기능
   
5. 문의 CTA
   - 견적 문의 버튼
   - 전화 문의 정보
```

#### 솔루션 페이지 목록
1. **스마트 현장안전 시스템** (/solutions/smart-safety.html)
   - 하위: 스마트 안전교육 시스템, 방문자 시스템
   
2. **타워크레인 통합안전 시스템** (/solutions/tower-crane.html)
   
3. **AI 영상 방송 관제시스템** (/solutions/ai-surveillance.html)
   - 하위: 지능형 영상·방송 관제 시스템
   
4. **스마트 환경센서 시스템** (/solutions/environment-sensor.html)
   - 하위: 환경센서 시스템, 초음파 풍향/풍속 시스템
   
5. **스마트 출입통제 시스템** (/solutions/access-control.html)
   - 하위: 안면인식 출입 통제 시스템, 스마트 통합 차량 관제 서비스

## 4. 글로벌 요소

### 4.1 헤더 (Header)
```
구조:
[로고] [메인 메뉴] [검색] [문의하기 버튼]

메인 메뉴:
- 회사소개
- 현장 안전 플랫폼
- 스마트 관제·제어
- 환경·출입관리
- 고객지원

드롭다운 메뉴:
- 마우스 오버 시 하위 메뉴 표시
- 부드러운 애니메이션 (0.2s fade-in)

모바일:
- 햄버거 메뉴
- 전체 화면 오버레이
- 아코디언 형태 하위 메뉴
```

### 4.2 푸터 (Footer)
```
구조:
┌─────────────────────────────────────────┐
│ [회사 로고 + 소개]  [사이트맵]  [문의]  │
│                                         │
│ [SNS 링크]                              │
│                                         │
│ [저작권 정보] [이용약관] [개인정보처리방침] │
└─────────────────────────────────────────┘

포함 내용:
- 회사명, 주소, 대표자
- 사업자등록번호
- 전화, 팩스, 이메일
- 주요 페이지 링크
- SNS 아이콘
- 저작권 표시
- 패밀리 사이트 (있는 경우)
```

### 4.3 플로팅 요소
```
- 맨 위로 버튼 (스크롤 시 표시)
- 빠른 문의 버튼 (우측 하단 고정)
  - 클릭 시 문의 폼 모달
  - 또는 전화 연결 (모바일)
```

### 4.4 검색 기능
```
- 헤더 검색 아이콘 클릭
- 전체 화면 검색 오버레이
- 자동완성 제안
- 최근 검색어
- 인기 검색어
```

## 5. 정보 아키텍처 원칙

### 5.1 콘텐츠 우선순위

#### 1순위: 핵심 비즈니스 정보
- 제품/솔루션 정보
- 프로젝트 실적
- 문의/견적 요청

#### 2순위: 신뢰 구축 정보
- 회사 소개
- 인증 및 특허
- 고객사

#### 3순위: 지원 정보
- FAQ
- 기술 자료
- 다운로드

#### 4순위: 부가 정보
- 공지사항
- 채용 정보

### 5.2 사용자 경로 최적화

#### 신규 방문자 경로
```
홈 → 사업분야/제품 → 실적 사례 → 문의
```

#### 재방문자 경로
```
홈 → 지원센터 → 자료 다운로드
```

#### 구매 결정권자 경로
```
홈 → 회사소개 → 실적 → 문의
```

#### 현장 관리자 경로
```
제품 검색 → 제품 상세 → 기술 자료 → 문의
```

### 5.3 정보 접근성

#### 3클릭 룰
- 모든 주요 정보는 홈에서 3클릭 이내 도달
- 깊이보다 넓이 우선 구조

#### 다중 진입점
- 같은 정보에 여러 경로로 접근 가능
- 관련 콘텐츠 링크 제공
- 브레드크럼 네비게이션

#### 명확한 레이블
- 사용자 언어로 메뉴 명명
- 전문 용어 최소화
- 일관된 용어 사용

## 6. 콘텐츠 전략

### 6.1 콘텐츠 유형별 관리

#### 정적 콘텐츠
- 회사 소개, 사업 분야, 제품 정보
- 연 1-2회 업데이트
- 버전 관리 필요

#### 동적 콘텐츠
- 공지사항, 프로젝트 사례
- 월 2-4회 업데이트
- CMS 관리

#### 다운로드 자료
- PDF 카탈로그, 매뉴얼
- 분기별 업데이트
- 버전 관리 및 아카이빙

### 6.2 콘텐츠 작성 가이드

#### 제목
- 명확하고 구체적
- 키워드 포함
- 50자 이내

#### 본문
- 역피라미드 구조 (결론 먼저)
- 짧은 문단 (3-4문장)
- 불릿 포인트 활용
- 시각 자료 포함

#### CTA
- 명확한 행동 지시
- 눈에 띄는 위치
- 하나의 주요 CTA

### 6.3 SEO 고려사항

#### 페이지별 메타 정보
```
Title: [페이지명] - [회사명] (60자 이내)
Description: [페이지 설명] (160자 이내)
Keywords: [주요 키워드 5-7개]
```

#### URL 구조
```
/about/company
/business/safety-solution
/products/category/product-name
/portfolio/project-name
/support/faq
```

#### 내부 링크
- 관련 페이지 상호 링크
- 앵커 텍스트 최적화
- 사이트맵 제공
