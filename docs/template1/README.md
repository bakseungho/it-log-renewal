## 프로젝트 주요 특징

### 1. 4단계 히어로 스토리텔링
- **슬라이드 1**: 브랜드 가치 (슬로건)
- **슬라이드 2**: 핵심 기술 (AI/통합안전)
- **슬라이드 3**: 현장 맞춤 솔루션 (환경센서)
- **슬라이드 4**: 신뢰와 유지보수 (CS)

### 2. 4대 핵심 솔루션
1. **현장 안전 플랫폼**
   - 스마트 현장안전 시스템
   - 타워크레인 통합안전 시스템

2. **스마트 관제·제어**
   - AI 영상 방송 관제시스템

3. **환경·출입관리**
   - 스마트 환경센서 시스템
   - 스마트 출입통제 시스템

### 3. 신뢰 구축 요소
- 숫자로 보는 신뢰 섹션 (설치 횟수, 연구 기간, 고객사 수)
- 주요 고객사 CI 로고
- 시공사례 중심의 실적 제시
- 인증 및 특허 정보

## 기술 스택

### 프론트엔드
- HTML5 (시맨틱 태그)
- CSS3 (Flexbox, Grid, CSS Variables)
- JavaScript (ES6+)

### 라이브러리 (CDN)
- **Swiper.js**: 히어로 슬라이더
- **GSAP**: 애니메이션 (스크롤 애니메이션, 카운트업)
- **카카오맵 API**: 오시는 길 지도

### 개발 규칙
- 외부 라이브러리는 CDN 방식 활용
- 애니메이션은 GSAP로만 구현
- 이미지 공간은 더미 링크로 적용 (이미지만 넣으면 되는 구조)

## 성공 지표 (KPI)

### 비즈니스 지표
- 문의 전환율 35% 향상
- 월 평균 방문자 수 25% 증가
- 평균 세션 시간 3분 30초 이상
- 페이지뷰 세션당 4.5페이지 이상

### 기술 지표
- 페이지 로딩 속도 2.5초 이내
- Lighthouse 점수 90점 이상
- Core Web Vitals Good 등급

### 사용자 경험 지표
- 이탈률 35% 이하
- 모바일 트래픽 55% 이상
- 재방문율 35% 이상
- 문의 폼 완료율 70% 이상
