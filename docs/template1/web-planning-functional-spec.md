# 웹 기획 기능 명세서

## 문서 개요

**프로젝트명**: (주)아이티로그 홈페이지 리뉴얼  
**작성일**: 2024  
**작성자**: Web Planner  
**버전**: 1.0

---

## 1. 기능 명세 개요

### 1.1 기술 제약사항
- **정적 웹사이트**: 서버 사이드 로직 없음
- **클라이언트 사이드**: JavaScript로 동적 기능 구현
- **외부 서비스 연동**: 문의 폼, 지도 API 등

### 1.2 기능 우선순위
- **P0 (Must Have)**: 필수 기능, Phase 1 구현
- **P1 (Should Have)**: 중요 기능, Phase 1-2 구현
- **P2 (Nice to Have)**: 선택 기능, Phase 2 이후

## 2. 공통 기능

### 2.1 헤더 (Header)

**기능 요구사항:**

| 기능 | 설명 | 우선순위 | 인터랙션 |
|------|------|---------|---------|
| 로고 | 클릭 시 메인 페이지 이동 | P0 | 링크 |
| GNB 메뉴 | 1Depth 메뉴 표시 | P0 | 호버 시 드롭다운 |
| 드롭다운 | 2Depth 메뉴 표시 | P0 | 마우스 오버/아웃 |
| 검색 아이콘 | 클릭 시 검색 오버레이 | P1 | 모달 열기 |
| 문의하기 버튼 | 문의 페이지 이동 | P0 | 링크 |
| Sticky 헤더 | 스크롤 시 상단 고정 | P1 | 스크롤 이벤트 |

**데스크톱 인터랙션:**
```
1. 메뉴 호버 → 드롭다운 표시 (0.2s fade-in)
2. 드롭다운 영역 벗어남 → 드롭다운 숨김 (0.2s fade-out)
3. 스크롤 100px 이상 → 헤더 배경 불투명, 그림자 추가
```

**모바일 인터랙션:**
```
1. 햄버거 아이콘 클릭 → 전체 화면 메뉴 오픈 (슬라이드 인)
2. 1Depth 클릭 → 2Depth 아코디언 펼침/접힘
3. 닫기(X) 또는 배경 클릭 → 메뉴 닫힘 (슬라이드 아웃)
```

**기능 명세:**

| 기능 | 설명 | 우선순위 |
|------|------|---------|
| 문의 폼 | 이름, 회사명, 연락처, 이메일, 내용 | P0 |
| 입력 검증 | 필수 항목 체크, 형식 검증 | P0 |
| 제출 | 이메일 전송 (외부 서비스) | P0 |
| 성공 메시지 | 제출 완료 알림 | P0 |
| 에러 처리 | 실패 시 에러 메시지 | P0 |

**입력 필드:**

| 필드 | 타입 | 필수 | 검증 |
|------|------|------|------|
| 이름 | text | O | 2자 이상 |
| 회사명 | text | O | 2자 이상 |
| 연락처 | tel | O | 숫자, 하이픈 |
| 이메일 | email | O | 이메일 형식 |
| 제목 | text | O | 5자 이상 |
| 내용 | textarea | O | 10자 이상 |
| 개인정보 동의 | checkbox | O | 체크 필수 |

**기술 구현:**
- 이메일 전송: FormSpree, EmailJS, 또는 Netlify Forms
- 검증: HTML5 validation + JavaScript
- 스팸 방지: reCAPTCHA (선택)

**인터랙션:**
```
1. 필드 입력 → 실시간 검증 (포커스 아웃 시)
2. 제출 버튼 클릭 → 전체 검증
3. 검증 실패 → 에러 메시지 표시 (빨간 테두리)
4. 검증 성공 → 로딩 인디케이터 표시
5. 전송 완료 → 성공 메시지 표시
6. 전송 실패 → 에러 메시지 + 재시도 안내
```
