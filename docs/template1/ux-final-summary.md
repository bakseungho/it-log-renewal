**변경 이유:**

### 2.2 솔루션 페이지 구조 변경

**변경 전:**
- 제품 중심 구조
- 제품 상세 페이지 개별 작성

**변경 후:**
- 솔루션 중심 구조
- 6개 솔루션 페이지
- 각 솔루션 내 하위 시스템 포함

**6개 솔루션 페이지:**
1. 스마트 현장안전 시스템 (하위: 안전교육, 방문자)

4. 스마트 환경센서 시스템 (하위: 환경센서, 초음파 풍향/풍속)
5. 스마트 출입통제 시스템 (하위: 안면인식, 차량 관제)

### 9.1 이미지

**히어로 이미지:**
- 크기: 1920x1080px
- 형식: WebP (폴백: JPG)
- 4개 슬라이드용

**솔루션 이미지:**
- 크기: 800x600px (4:3 비율)
- 형식: WebP (폴백: JPG)
- 각 솔루션별 대표 이미지

**시공사례 이미지:**
- 크기: 600x338px (16:9 비율)
- 형식: WebP (폴백: JPG)
- 각 솔루션별 3-4개

**고객사 로고:**
- 크기: 200x100px
- 형식: PNG (투명 배경)
- 16개 이상

**인증서/특허:**
- 크기: 원본 스캔 (고해상도)
- 형식: PNG 또는 JPG
- 8개 이상

### 9.2 아이콘

**필요 아이콘:**

- 체크, 경고, 오류
- 솔루션별 대표 아이콘 (4개)

**형식:**
- SVG (벡터)
- 24x24px 기본 크기

### 9.3 로고
