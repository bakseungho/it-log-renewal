## 1. 프로젝트 배경 및 목표

### 1.1 회사 소개
- **회사명**: (주)아이티로그
- **사업 분야**: 건설 현장 제품 솔루션 설치 및 시공
- **주요 솔루션**:
  - 현장 안전 플랫폼 (스마트 현장안전, 타워크레인 통합안전)
  - 스마트 관제·제어 (AI 영상 방송 관제)
  - 환경·출입관리 (환경센서, 출입통제)
- **주요 고객**: 건설사, 시공사, 공공기관
- **비즈니스 특성**: B2B, 고관여 의사결정, 신뢰 중심

### 1.2 프로젝트 목표
1. **비즈니스 목표**
   - 브랜드 신뢰도 향상 및 전문성 강조
   - 온라인 문의 전환율 30% 증가
   - 정보 탐색 효율성 50% 향상
   - 모바일 사용자 경험 대폭 개선

2. **사용자 목표**
   - 필요한 정보를 빠르게 찾을 수 있는 직관적인 인터페이스
   - 솔루션별 상세 정보 및 시공 사례를 쉽게 확인
   - 간편한 문의 및 원격지원 요청 프로세스

3. **디자인 목표**
   - 전문적이면서도 접근하기 쉬운 디자인
   - 신뢰감을 주는 비주얼 아이덴티티
   - 모바일 최적화 및 반응형 디자인
   - 접근성 준수 (WCAG 2.1 AA)

#### 3.2.2 디자인 시스템 구축 (Week 4 후반)
**목표**: 일관성 있는 디자인을 위한 디자인 시스템 구축

**작업 내용**:
1. **컬러 팔레트**
   - Primary Color: 메인 브랜드 컬러 (CTA, 링크, 강조)
   - Secondary Color: 보조 컬러
   - Accent Color: 강조 컬러
   - Neutral Colors: 배경, 텍스트, 테두리
   - Semantic Colors: Success, Error, Warning

2. **타이포그래피 시스템**
   - Font Family: 한글 (Noto Sans KR, Pretendard), 영문 (Roboto)
   - Font Sizes: H1-H6, Body, Small
   - Font Weights: Regular (400), Medium (500), Bold (700)
   - Line Height: 1.5-1.8
   - Letter Spacing

3. **간격 시스템**
   - Spacing Scale: 8px 기준 (8, 16, 24, 32, 48, 64, 80px)
   - Section Spacing: 80px (Desktop), 48px (Mobile)
   - Component Spacing: 40px (Desktop), 24px (Mobile)

4. **UI 컴포넌트 라이브러리**
   - **버튼**: Primary, Secondary, Text, Large, Full Width
   - **입력 필드**: Input, Textarea, Select, Checkbox, Radio
   - **카드**: Product Card, Project Card, Info Card
   - **네비게이션**: Header, Footer, Breadcrumb, Pagination
   - **모달/팝업**: Alert, Confirm, Lightbox
   - **기타**: Badge, Tag, Tooltip, Loading Spinner

5. **그리드 시스템**
   - Desktop: 12 컬럼, 1200px 최대 너비
   - Tablet: 8 컬럼
   - Mobile: 4 컬럼

6. **아이콘 세트**
   - 스타일: Line 또는 Solid (일관성 유지)
   - 크기: 16px, 24px, 32px, 48px
   - 소스: Font Awesome, Heroicons, 또는 커스텀

**도구**: Figma (컴포넌트 라이브러리)

**산출물**: `docs/ux-design-system.md` (Figma 링크 포함)

## 4. 작업 가이드라인

### 4.1 디자인 원칙
1. **신뢰와 전문성**: B2B 특성상 신뢰감을 주는 디자인
2. **명확성**: 정보 계층이 명확하고 이해하기 쉬운 레이아웃
3. **일관성**: 전체 사이트에서 일관된 디자인 언어
4. **접근성**: 모든 사용자가 쉽게 사용할 수 있는 디자인
5. **효율성**: 사용자가 빠르게 목표를 달성할 수 있는 UX

### 4.2 B2B 웹사이트 디자인 특징
- **데이터 중심**: 구체적인 수치 및 실적 강조
- **신뢰 요소**: 고객사 로고, 인증서, 프로젝트 실적 전면 배치
- **전문성**: 깔끔하고 정돈된 레이아웃, 고품질 이미지
- **명확한 CTA**: 각 페이지마다 명확한 행동 유도
- **정보 밀도**: 모바일에서는 간결하게, 데스크톱에서는 상세하게

### 4.3 반응형 디자인 가이드
**브레이크포인트**:
- Mobile: 320px - 767px
- Tablet: 768px - 1023px
- Desktop: 1024px - 1440px
- Large Desktop: 1441px+

**디바이스별 전략**:
- **모바일**: 1 컬럼, 세로 스택, 터치 영역 최소 44x44px
- **태블릿**: 2 컬럼, 일부 콘텐츠 가로 배치
- **데스크톱**: 3-4 컬럼, 풍부한 정보 제공

### 4.4 접근성 가이드 (WCAG 2.1 AA)
- **색상 대비**: 텍스트와 배경 대비 4.5:1 이상
- **터치 영역**: 최소 44x44px
- **키보드 네비게이션**: 모든 기능 키보드로 조작 가능
- **포커스 인디케이터**: 명확한 포커스 표시
- **대체 텍스트**: 모든 이미지에 의미 있는 alt 속성
