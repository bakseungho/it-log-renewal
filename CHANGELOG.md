## 버전 관리 규칙

이 프로젝트는 [Semantic Versioning](https://semver.org/)을 따릅니다:

- **MAJOR** (1.x.x): 호환되지 않는 API 변경
- **MINOR** (x.1.x): 하위 호환성을 유지하는 기능 추가
- **PATCH** (x.x.1): 하위 호환성을 유지하는 버그 수정

## 변경사항 카테고리

- **추가됨**: 새로운 기능
- **변경됨**: 기존 기능의 변경
- **수정됨**: 버그 수정
- **제거됨**: 제거된 기능
- **보안**: 보안 관련 수정
- **deprecated**: 곧 제거될 기능
